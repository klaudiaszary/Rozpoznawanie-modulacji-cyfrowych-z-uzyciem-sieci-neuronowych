# Rozpoznawanie modulacji cyfrowych z użyciem sieci neuronowych

Projekt dotyczący automatycznej klasyfikacji modulacji cyfrowych na podstawie obrazów diagramów konstelacji I/Q z wykorzystaniem głębokich sieci neuronowych.

W ramach projektu przygotowano własny zbiór danych obejmujący 17 rodzajów modulacji cyfrowych oraz różne warunki transmisyjne i zakłócenia sygnału. Przeprowadzono trening i porównanie kilku architektur sieci neuronowych, m.in. ResNet18, ResNet50, EfficientNet-B0 oraz Vision Transformer.

Najlepszy model ResNet18 osiągnął dokładność 89,55% na zbiorze testowym oraz 83,33% podczas weryfikacji na rzeczywistych diagramach konstelacji.
